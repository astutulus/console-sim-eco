# console-sim-eco

C++ | Console | ASCII graphics | Simulated ecosystem

## Summary
- Rabbit's energy continually depleted
- Rabbit moves towards grass; eats to restore energy
- If energy runs out, program ends
- User must keep scattering food

## Control

```
              G         - sow grass
```
